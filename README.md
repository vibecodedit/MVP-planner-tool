## Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd MVP-planner-tool
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:12000`

## How to Use

### 1. Start with Your Current Task
- The main screen shows your current task with full details
- Click the circle icon or "Mark as Complete" button to finish tasks
- Use arrow buttons to navigate between tasks

### 2. View All Phases
- Click "Overview" to see all 6 development phases
- Each phase shows progress and all tasks within it
- Click any task to jump directly to it

### 3. Customize Your Tasks
- Click the settings icon to open the task editor
- Modify existing tasks or add new ones
- Adjust priorities and descriptions to match your project

### 4. Track Your Progress
- The progress bar shows real-time completion percentage
- Each phase has its own progress indicator
- 100% means you're ready to deploy to alpha testers!
